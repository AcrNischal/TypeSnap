How to Use TypeSnap

Creating Snippets

Click "New Snippet" to create a text expansion shortcut. Triggers automatically start with /.

  • Trigger: The short code you type (e.g., /em).
  • Content: The text or URL that replaces the trigger.

Omnibox Search

Type pt followed by a space in your browser's address bar to search snippets.

  • Select a snippet and press Enter to paste or navigate.
  • If the snippet is a URL, it will open automatically.

Spotlight Overlay

Press Ctrl + / (or Cmd + / on Mac) on any page to open Spotlight.

  • Type to search your snippets.
  • URL Snippets: Press Enter to navigate immediately.
  • Text Snippets: Press Enter to copy to clipboard.

About the Project

TypeSnap is designed to boost your productivity by allowing you to create custom text snippets and expand them instantly on any website. Built with privacy in mind, all your data is stored locally on your device.
